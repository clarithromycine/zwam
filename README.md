# ZWAM

ZWAM is a WhatsApp-WAM (with w:stats schema) inspector which including 5000+ stat/log items now.

### WHAT IS WAM ?
WAM stand for "WhatsApp Analytics & Metrics" 
WAM is WhatsAppâ€™s internal telemetry and event-logging system used to collect, encode, and transmit client-side analytics data to WhatsApp servers.
The WAM system is designed to:
- Monitor app behavior and feature usage
- Measure performance, reliability, and latency
- Support A/B experiments and feature rollouts
- Assist in anti-abuse, risk scoring, and quality monitoring

### REQUIREMENTS
1. Rooted ANDROID phone
3. [Frida server ](https://frida.re/docs/android/)  installed on the phone  ( version 16.7.19)
4. Both Whatsapp and Whatsapp business installed on the phone **from the Play Store**

### HOW TO INSTALL
- cd android
- npm install

### HOW TO RUN
- python run.py
