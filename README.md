# ZWAM

ZWAM is a WhatsApp-Wam (with w:stats schema) inspector which including 5000+ stat/log items now.

### REQUIREMENTS
1. Rooted ANDROID phone
3. [Frida server ](https://frida.re/docs/android/)  installed on the phone  ( version 16.7.19)
4. Both Whatsapp and Whatsapp business installed on the phone **from the Play Store**

### HOW TO INSTALL
- cd android
- npm install

### HOW TO RUN
- python run.py
